## üöÄ Getting Started

To run this application locally, follow these steps:

### 1. Obtain an API Key

1. Visit [OpenWeatherMap](https://openweathermap.org).
2. Create a free account and navigate to the **API keys** section.
3. Generate a new key and copy it to your clipboard.
4. **Note**: It may take up to 2 hours for a new key to be activated by OpenWeather.

### 2. Configuration

1. Open the `script.js` file.
2. Locate the `apiKey` variable at the top of the file.
3. Replace `'YOUR-API-KEY'` with the key you generated.

### 3. Launch

1. Simply open `index.html` in any modern web browser.
2. Allow **Location Access** when prompted to enjoy the automatic weather detection feature.
